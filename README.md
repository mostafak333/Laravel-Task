## About Project

This is laravel project about Goldady stock,

## Informations

The project run on localhost: 127.0.0.1, port: 8000,
DB port:3306

## Important instructions to run the project

-   Create DB in phpmyadmin
-   Check this in .env file in the project
    -   DB_CONNECTION=mysql
    -   DB_DATABASE=name of DB you created
    -   DB_USERNAME=your username
    -   DB_PASSWORD=your password
-   Run: php artisan migrate to create all tables in DB
-   Run: php artisan serve to run project in localhost
-   Enjoy!
